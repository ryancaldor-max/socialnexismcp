# SocialNexis Post Tools (Desktop Extension)

AI detector and humanizer for LinkedIn/X posts, post quality analyzer, and
structured post idea generator. Free, no API key required.

Tools: `check_post_tells`, `analyze_post`, `generate_post_ideas`. Full API
reference: https://socialnexis.com/mcp/docs

## Privacy Policy

This extension sends the post text or topic you submit to the SocialNexis
API (api.socialnexis.com) for processing. No API key, no account, and no
stored content: requests are processed and counted, not retained. No data is
shared with third parties beyond the model inference required to produce the
result. Full policy: https://socialnexis.com/privacy
